# Test Lambda Works - Gestión de Contactos

Una aplicación web simple para la gestión de contactos. Se aceleró el desarrollo de este proyecto usando **Antigravity**.

## Stack Tecnológico

Este proyecto fue desarrollado utilizando las siguientes tecnologías:

- **Frontend:** React (desarrollado con Vite)
- **Backend:** Node.js con Express
- **Base de Datos:** SQLite
- **ORM:** Sequelize

## Instrucciones de Instalación y Ejecución

Para correr este proyecto, debemos iniciar primero el backend y luego el frontend:

### 1. Iniciar el Backend

Abrí una terminal en la carpeta del proyecto y luego ingresá a la carpeta del backend. Instalá las dependencias y ejecutá el servidor:

```bash
cd backend
npm install
node index.js
```

### 2. Iniciar el Frontend

Abrí otra terminal dejando que el backend se ejecute en segundo plano, ingresá a la carpeta del frontend, instalá las dependencias y arrancá el entorno de desarrollo:

```bash
cd frontend
npm install
npm run dev
```
Esto iniciará la aplicación de React y nos dará una dirección local (por defecto es `http://localhost:5173`).
